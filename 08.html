# Vue 플러그인 장점

// router/index.js
import Vue from "vue";
import VueRouter from "vue-router";

Vue.use(VueRouter);

Vue-router는 공식 플러그인이라 위와 같은 방법으로 사용 선언한다.

// myplugin.js
export default {
    install() {
        console.log("플러그인 설치");
        
    }
}

// main.js
import MyPlugin from './plugins/MyPlugin'

Vue.use(MyPlugin);

플러그인은 초기에 실행할 때 한 번만 실행되므로, 코드가 중복되지 않는 장점이 있다.

main.js가 라이브러리의 속성을 파악하기 쉽게 플러그인화 시키면 좋다.
